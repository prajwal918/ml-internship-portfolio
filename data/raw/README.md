# Raw Dataset Placement

Put downloaded datasets here. The app automatically uses these files when present and falls back to generated demo data when they are missing.

## Movie Recommendation

Supported:

- MovieLens `ratings.csv` and `movies.csv`
- MovieLens 100K `u.data` and `u.item`

Download: https://grouplens.org/datasets/movielens/100k/

## Customer Churn

Supported:

- `Telco-Customer-Churn.csv`
- Any CSV with `churn` in the filename and IBM/Telco columns:
  `tenure`, `MonthlyCharges`, `TotalCharges`, `Contract`, `InternetService`, `PaymentMethod`, `Churn`

Download: https://www.kaggle.com/datasets/blastchar/telco-customer-churn

## Fake News

Supported:

- `Fake.csv` and `True.csv`
- A generic news CSV with text columns such as `text`, `statement`, `content`, `article`, or `title`, plus label columns such as `fake`, `label`, `target`, or `class`

Download: https://www.kaggle.com/datasets/clmentbisaillon/fake-and-real-news-dataset

## Employee Performance

Supported:

- IBM HR Analytics CSV containing `PerformanceRating`
- Expected columns include `Age`, `YearsAtCompany`, `TrainingTimesLastYear`, `Department`, `JobLevel`, `Gender`

Download: https://www.kaggle.com/datasets/pavansubhasht/ibm-hr-analytics-attrition-dataset

## Stock Market

Supported:

- `stock_ohlcv.csv` with `date`, `ticker`, `open`, `high`, `low`, `close`, `volume`

Generated by:

```powershell
python scripts/download_public_data.py
```
